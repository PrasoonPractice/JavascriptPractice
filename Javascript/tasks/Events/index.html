Event Handling in Javascript

An Event is the change in the state of an object.
There are various events in HTML which represents that some activity is performed by the user or by the browser.
The process of reacting over the events is called Event Handling.
The Javascript handles the HTML events via Event Handlers.

The following table shows some of the HTML events and their event handlers.
These are also applied on the table as;

  • The first column has mouseover, mouseout and click events.
  • The second column has mousedown and mouseup events.
  • Only the first cell in the third column has mousemove event.

EventEvent HandlerDiscription
click onclick() When mouse click on an element
mouseover onmouseover() When the cursor of the mouse comes over the element
mouseout onmouseout() When the cursor of the mouse leaves an element
mousedown onmousedown() When the mouse button is pressed over the element
mouseup onmouseup() When the mouse button is released over the element
mousemove onmousemove() When the mouse movement takes place.
